# mip-mmtpk

mip-mmtpk 网站自用业务组件

标题|内容
----|----
类型|业务
支持布局|N/S
所需脚本|https://mipcache.bdstatic.com/static/v1/mip-mmtpk/mip-mmtpk.js

## 示例

### 基本用法
```html
<mip-mmtpk phpurl="url" aid="aid" now="num"></mip-mmtpk>
```

## 属性

### phpurl

说明：调用脚本的url地址
必选项：是
类型：字符串

### aid

说明：文章id
必选项：否
类型：字符串

### num

说明：文章当前分页号
必选项：否
类型：字符串
